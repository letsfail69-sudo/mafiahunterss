# REVIZE WIKI

- Normalizována adresářová struktura ZIPu (lomítka `/`).
- Opraveno a validně znovu sestaveno 72 HTML stránek pomocí HTML5 parseru.
- Všechny tabulky jsou v posuvném responzivním obalu.
- Přidán samostatný mobilní navigační přepínač; obsah článku je na mobilu zobrazen před menu.
- Přidány robustní mobilní styly bez změny původního vizuálního stylu.
- Obrázky bossových zbraní používají skutečné PNG soubory a nejsou líně načítané.
- Přidáno verzování CSS/JS pro omezení problémů s cache prohlížeče a GitHub Pages.
- Zachováno ukládání otevřených sekcí menu v localStorage.
